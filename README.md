# Modelos moleculares

[Início][inicio] › [Modelos][modelos]

Note que, para alguns modelos, as *conformações* mais comuns estão disponíveis.  
*Mapas de Potencial Eletrostático* (MPE) de alguns modelos selecionados também estão disponíveis.
